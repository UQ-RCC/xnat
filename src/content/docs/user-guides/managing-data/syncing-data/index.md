The UQ-RCC XNAT may be able to sync datasets to other XNATs in Australia. This is set up by RCC support on request.

## Request data sync

To check whether your project can be synced, send a ticket to RCC support (rcc-support@uq.edu.au) with:

- Your XNAT project name
- The details of the XNAT you wish to share data with

Support will confirm whether syncing is possible and arrange it for you.
