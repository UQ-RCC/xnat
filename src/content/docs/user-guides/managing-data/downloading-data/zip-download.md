---
title: Zip Download
description: Downloading data as Zip using the XNAT UI
---

The Project, Subject and Sessions page has the option to **Download Images** from the right hand side menu

![Download Images menu](./zip-download/downloading-data-1.png)
![Download options](./zip-download/downloading-data-2.png)
![Download dialog](./zip-download/downloading-data-3.png)

Select **Option 2: ZIP download**

:::caution[Note]
Untick **Simply downloaded archive structure** to keep scan names in scan folders
:::

![ZIP download options](./zip-download/downloading-data-zip.png)

Filter out the Session and scans from the first two columns and select **Submit**
